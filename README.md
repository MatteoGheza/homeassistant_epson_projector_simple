# Epson Projector Simple - Home Assistant Custom Component

This custom integration connects a Epson Projector using TCP protocol.

## Notes

- AI was used to help convert the original library into this Home Assistant custom component.
- Testing and validation were performed manually.
- Tested only with `PureTools PT-MA-HD44M 4x4 4K HDMI Matrix`.
